◆ testPassengerCountryCode()

void es.ull.passengers.PassengerTest.testPassengerCountryCode ( )

Test para comprobar la validación del código de país en el constructor.

En este test se comprobará el algoritmo de validación del código ISO de país correspondiente a la máquina local. De este modo, aquel pasajero creado con un código de país distinto a España (en este caso) deberá ocasionar el lanzamiento de una excepción con el mensaje de "Invalid country code". En otro caso, el pasajero ha de poder crearse sin ningún problema.

◆ testToString()

void es.ull.passengers.PassengerTest.testToString ( )

Test para la comprobación del método toString del pasajero.

En este test se comprobará que todos los toString invocados para cualquier objeto passenger, devuelvan siempre las propiedades 'name', 'identifier' y 'countryCode'.
